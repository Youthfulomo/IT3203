HTTP/0.9

HTTP/0.9 was the first version of HTTP. It was released in 1991 and could only request simple HTML pages. It did not support headers, status codes, or many of the features used today.

HTTP/1.0

HTTP/1.0 introduced headers, status codes, and support for different types of files. It made the web much more useful but still opened a new connection for every request.

HTTP/1.1

HTTP/1.1 improved performance by allowing multiple requests over the same connection. It quickly became the standard version used across the internet for many years.

HTTP/2

HTTP/2 introduced multiplexing, allowing several requests to travel over one connection at the same time. This greatly improved the speed of modern websites.

HTTP/3

HTTP/3 is the newest version of the protocol. It uses QUIC instead of TCP, making websites faster, more reliable, and more secure while also reducing delays caused by lost network packets.

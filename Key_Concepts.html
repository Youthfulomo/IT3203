1. HTML

HTML stands for HyperText Markup Language. It is the standard language used to create the structure of webpages.

2. CSS

CSS stands for Cascading Style Sheets. It is used to change the look of a webpage by adding colors, fonts, spacing, and layouts.

3. Hyperlinks

Hyperlinks allow users to move from one webpage to another by clicking on text or images.

4. HTML Tags

HTML tags tell the browser how to display content. Most tags have an opening tag and a closing tag.

5. HTML Elements

HTML elements are the building blocks of a webpage. An element usually includes a tag and the content inside it.

6. Attributes

Attributes provide additional information about an HTML element. For example, the href attribute tells a link where to go.

7. Web Browser

A web browser is a program that reads HTML files and displays webpages. Examples include Google Chrome, Microsoft Edge, and Firefox.

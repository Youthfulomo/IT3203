Understanding HTTP

Welcome

Welcome to my IT3203 project website. This website explains HyperText Transfer Protocol (HTTP), how it works, and how it has changed over time. It also includes a quiz to test your understanding of the material.

Website Overview

The HTTP Basics page explains how HTTP works using the client-server model. The HTTP Versions page compares each version of HTTP from HTTP/0.9 through HTTP/3. The Key Concepts page reviews important networking and web development terms. The Quiz page allows visitors to test their knowledge, while the References page lists the resources used throughout the project. The About Me page provides information about the author.

Why HTTP Matters

Every time a user visits a website, HTTP allows a web browser and a web server to communicate. Modern versions of HTTP improve website speed, security, and reliability, making them an essential part of today's Internet.
